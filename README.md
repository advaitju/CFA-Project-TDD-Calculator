## About
A simple calculator app created to trial the TDD process (Fail test, Pass test, Refactor code) using test-unit gem.
## Installation
1. Clone repo
2. Run commands in cloned repo directory
```
gem install bundler
bundler install
ruby test_calculator.rb
```
